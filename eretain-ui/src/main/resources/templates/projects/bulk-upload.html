Required Columns:
  • Project Code — Unique identifier (e.g., PRJ-001)
  • Project Name — Name of the project
  • Project Type — FIXED_PRICE, TIME_AND_MATERIAL, RETAINER, INTERNAL, SUPPORT
  • Billability Category — BILLABLE, NON_BILLABLE, INTERNAL, INVESTMENT
Optional Columns:
  • Description — Project description
  • Status — PROPOSED (default), ACTIVE, ON_HOLD, COMPLETED, CANCELLED
  • Client Name — Client/customer name
  • Start Date — Format: yyyy-MM-dd (e.g., 2026-04-01)
  • End Date — Format: yyyy-MM-dd (e.g., 2026-12-31)
  • Budget — Numeric value (e.g., 100000.00)
  • Currency — 3-letter code (default: USD)
